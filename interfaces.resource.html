ref(path, query=None, fragment=None)¶

Returns a new resource referencing the given path relative to this resource path.

The new resource will have the same scheme and location than this resource.

Parameters:
  • path (string) – Relative path.
  • query (string) – Optional query part.
  • fragment (string) – Optional fragment part.
Return type:
